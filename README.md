# Senwei Xie's academic homepage

This repository contains Senwei Xie's academic homepage, built with
[al-folio](https://github.com/alshedivat/al-folio) and deployed with GitHub
Pages.

## Local preview

Follow the al-folio installation guide, then run:

```bash
bundle exec jekyll serve
```

## GitHub Pages note

Because this repository is named `xiesenwei.github.io` under the `xsw1208`
account, the default GitHub Pages URL is:

```text
https://xsw1208.github.io/xiesenwei.github.io/
```

If the repository is renamed to `xsw1208.github.io` or a custom root domain is
configured, update `_config.yml` and set `baseurl:` to empty.
